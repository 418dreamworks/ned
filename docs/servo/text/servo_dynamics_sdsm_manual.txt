This manual is a general purpose manual describing the theory of
operation, product specification, installation instructions and
adjustment procedure, for the SDSMB 1625-17 Servo Amplifier. The
(ET) version can also use an encoder feedback and/or a tachometer
feedback signal. The discussion covering the encoder feedback
will be highlighted thru the text, in addition to the other
features listed in the specification sheet this model does not
require an isolation transformer and can derive its 170 VDC power
directly from a 120 VAC source.

The Servo Amplifier described herein is a Surface Mount Pulse
Width Modulated (PWM) type used for controlling brushless type

servo motors.

A complete chassis assembly consists of one or more amplifier
modules mounted on a sub panel, containing a bus power rectifier,
filter capacitor, fans and associated wiring.

DISCUSSION OF A BLOCK DIAGRAM

The amplifier is a Hall commutated current source PWM.

Signal enters at the top left corner of the diagram. Input is
differential. For a single-ended command, apply the signal to
the appropriate terminal and ground the other input terminal at
the ground the other input terminal at the signal source.

The standard tachometer input is single ended input.

The PWM modulator monitors current flow in the power bridge and
pulses the system accordingly .

The Hall sensors in the motor provide information regarding shaft
position and commutate the system. The Code is 120 degree, so
the two invalid codes are 000 and 111 and are the codes which
will be obtained if the sensors lose power or are not properly
connected.

If the code is invalid, the amplifier will be inhibited. If
there is an invalid transition between valid codes, the amplifier
momentarily shuts down the power bridge and restarts immediately
when a valid Hall code is present.

Fault conditions set latches and are indicated by red LED’. They
shut down both the Hall decoding and the drivers of the power
bridge. The Hall decoding remains disabled while the fault
latches are being reset thus preventing overriding of the fault
sensor during a continued fault condition.

+6 Volts nominal is provided for operating Halls sensors.

Differential input inverting signal -- This input is

Note: This pin can also be used as a single ended

Differential input non-inverting signal -- This input

Note: This pin can be used as a single-ended input.
However, Jl, Pin 4 must be grounded when J1,

buffer op-amp will be 1/2 its normal gain. If Jl
Pin 4 is not grounded additional noise can be
introduced in J1, Pin 5. Use Jl, Pin 2 as
common if Jl, Pin 5 is used as a single ended
input.

A DIFFERENTIAL OP-AMP PROVIDES THE ADDED BENEFITS OF BEING ABLE
TO REJECT A SIGNAL (WHICH IS PRESENT ON BOTH SIGNAL WIRES BECAUSE
OF ITS COMMON MODE REJECTION CAPABILITY). ALSO USING THE
DIFFERENTIAL INPUT REDUCES THE POSSIBILITY OF GROUND LOOPS.
Pin 6 - Tach input - Used for tach input only this input has
additional tach filtering and conditioning - single
ended.

Pin 7 - Option - Output from Encoder to Tach
If module has the Encoder to Tach optional feature and
Encoder to Tach is used as the velocity feedback to
close the velocity loop in the amplifier then connect
J1, Pin 7 to J1, Pin 6 to bring Encoder to Tach signal
into Tach input.

Pin 8 - Current Monitor Output - Unipolar (oneway) output +10
Volts out equals approximately 25 amps.

Note: This pin is the vector sum of the output currents and is
used internally for the RMS current measurement, hence it has a
non linear output. There is also an offset of about .3 Volts in
the 2 polarities of current because of the way the voltage is
generated.
Pin 9 - Enable input/Remote reset
This pin has to be pulled above 3 volts to enable the
amplifier. This pin is pulled to ground internally
through a 10K resistor and is normally low.
Do not pull this pin above +12 volts.
Removing the +5 volt power inhibits the amplifier.

Notes:

It is recommended that this pin be connected to an External +5
volt logic signal.

The green RUN led will extinguish when the amplifier is
inhibited.

The resetting of the amplifier is complete with a 1 mila second
delay, after PIN 9 is connected to the +5 volts. This sequence
of resetting is such that if PIN 9 is kept high, the safety
circuits can not be overridden.

If a Fault Condition is still present after reset, the amplifier
will not restart and the appropriate Fault red led will be lit.

When power is removed from the amplifier or system it is a good
idea to remove the +5 volts from Jl, Pin 9 in order to prevent
any erratic motor operation as the various logic supplies come
out of regulation.

J1, Pin 16 (Fault output) will stay high when amplifier is in
inhibited (when the +5 volts is removed for Jl Pin 9).
Pin 10 - Hall output voltage - Nominal +6 volt. This output
Pin has 15 MA of current capacity (not regulated) and
its output power can be used to provide power to the
halls.

Note: If the motor is supplied with an encoder with commutation
tracks and the encoder is powered by an external +5 volt supply
do not connect the output of Jl, Pin 10 to the encoder.

Pin 11 - Common

Pins 12 - Hall detector inputs - These pins are pulled up

13& 14 internally through a 10K resistor to a nominal 6
volts.

These hall inputs act as the electronic commutator
for indicating the rotors shaft position for proper
timing for the power to be delivered to the motor so
the motor can develop the proper torque.

Note: If the encoder which has commutation tracks is providing
the +5 power to the commutation signals for Jl Pins 12, 13 and 14
make sure the return for the +5 power is connected to J1, Pin 15
(common). If encoders commutation tracks are line driver output
use one output of each the appropriate channel.

Pin 1i5 - Common

Pin 16 - Fault Output - This pin will go low if a fault occurs
in the amplifier. This pin has an uncommitted
collector output. This pin is typically pulled up to
+12 Volts but can be pulled up to a maximum of +24
volts and has 5 milliamps of current capacity.

This pin will if pulled up to some voltage remain

high if the amplifier is inhibited (when +5 Volts is
removed from Jl, Pin 9).

J4 - Encoder Inputs - 5 Volt Logic (Option)

The J4 Pins are used as encoder inputs when the encoder to
tach feature is used. The encoder’s outputs maybe single
ended or line driver.

Pin 1 No Connection

Pin 2 Encoder input Channel B Pins 2 and 4 Normally
sit logic high
Pin 3 No Connection

Pin 4 Encoder input Channel A

Pin 5 Encoder Common

It is very important that Pin 5 encoder common by a separate wire
and in the same shielded cable as J4 Pins 2 and 4.

The encoder cable connected to J4 is to be treated as a digital
data cable otherwise unnecessary noise will enter the amplifier
on J4 and erratic motor operation will occur.

Also noise pickup on the cable can go the other direction and get
into the controller.

Notes: J4, Pins 2 and 4 are to be operated from a 5 volt
encoder. These 2 input Pins are connected to level
translators.

These encoder inputs have a +1.5 volt threshold and require

2.4 MA of sinking to pull the input pins to ground. However the
input pins will recognize a logic low at 1 MA of sink. The
encoder inputs should not be pulled above +8 volts.

Do not tie the shield of the encoder at the amplifier because the
encoders shield will be tied at the controllers’ ground.

Note: Make sure there is a jumper installed on J1 between Pins 6
and 7 so output of encoder is connected back in to the
tachs input.

Caution: Do not leave this jumper on if you are not using the
J4 (encoder inputs) otherwise very erratic and
unpredictable motor shaft motion will occur.

RMS CURRENT - This LED provides indication for these 2
conditions:

OVERTEMPERATURE of the heatsink - the sensor is set to trip

An overtemperature condition might exist, for the following

reasons:

a) Clogged or dirty cooling fans.

b) Insufficient air flow across the heatsink or cooling of

c) Ambient cabinet temperature too high.

Excessive RMS Current means the amplifier was required to

This condition can exist if a machine is asked to perform a

If a motor is mechanically stalled or binding.

If a motor has a shorted armature.
SURGE/GROUND FAULT

This LED provides indication for 2 conditions.

SURGE CURRENT - If there is an excessive amount of current
thru the power transistors in the output power bridge.

This condition can exist:
a) If an output power device has become damaged.
b) If the output leads to the motor have become
shorted together.
GROUND FAULT - This LED will give an indication if one
of the output wires to the motor is shorted
ground.

This condition can exist due to

a) Faulty or pinched wiring.
b) Motor arcing to its case ground.

The signal pot. (R95) is. used for the command
signal. Turning Rl “CW” increases the amount of
command input signal to the amplifier.

The tach. pot. (R94) is used for the tach. input.
This input should be used as the tach. input because
of special signal conditioning on this input.
Turning R2 “CW” increases the amount of tach-
feedback into the amplifier.

The CMP (compensation) pot. (R93) is used to
increase or decrease the response (bandwidth)
of the amplifier. Turning R3 “CW” increases
the response of the amplifier.

The current limit pot. (R92) is used to increase
or decrease the response (bandwidth) of the
amplifier. Turning R4 “CW” increases the output
current, to the motor, from zero amps to maximum
peak amps.

The balance pot. (R91) is used to adjust internal
offsets in the amplifier. The function of this pot
is such that for zero input volts the output current
should be at zero amps.

The RMS pot. (R90) is for changing the level of the
RMS current. The amplifier is capable of providing
maximum RMS current when R6 is fully “CW”. The
maximum current is approximately 0 amps.

INSTALLATION (See Figure 3)

The amplifier modules are supplied with their own specific
installation drawing. It is important that one follows the
instructions for proper connection of input power, input signals,
tachometer input or encoder input and motor outputs.

The amplifier modules should be installed in a location which is
suitable for electronic equipment.

The maximum ambient cabinet temperature should not exceed 50
degrees C, otherwise nuisance over-temperature tripping may
occur.

When installing an amplifier in a cabinet, make sure there is
adequate space around the amplifier to allow for proper cooling.

BEFORE CONNECTING THE INPUT POWER TO THE AMPLIFIER MEASURES THE
INPUT VOLTAGE WITH A VOLT METER.

The SDSMB(ET) amplifier high voltage bus supply can operate
directly from 120 VAC (grounded neutral) power sources.

If a transformer is used it can supply the needs of several
amplifier modules on a chassis provided it is of the proper size
and KVA rating.

Some transformers have dual primaries, check that the proper
input taps and jumpers are installed correctly.

Typically the output voltage of the main power transformer at the
secondary can range from 50 VAC RMS to a maximum of 120V AC RMS.
This voltage when rectified on the amplifier chassis becomes the
+DC Bus voltage.

115-125 VAC (2 Amp) 50/60HZ power is required for the cooling
fans.

The signal input (differential input recommended) is applied to
PIN 4 of connector J1 with respect to signal common PIN 5 of Jl.

Use shielded wire on the signal and tachometer input wires going
to the amplifier.

Shielded motor wires with one end of the shield grounded at the
amplifier end is recommended due to the generated RF of the
switching action in the amplifier.

If the motor inductance is less than the recommended minimum (see
spec. sheet), an external inductor must be used.

not grounded. Three pin power cords are grounded. Be sure to
use an isolator adaptor.

Make sure you do not ground the output section of the amplifier.
The output legs of the amplifier at rest, may be 85 VDC above

SIMPLIFIED SYSTEM “SET UP”
BEFORE APPLYING POWER

1) If a transformer is being used check to be sure all
transformer jumpers correct and the transformers are wired
for proper input voltages.

2) Verify all input voltages before connecting them to the
amplifier.

3) Be sure the amplifier is connected per the installation
drawing, supplied with the amplifier.

4) Make sure the machine has sufficient travel in case of an emergency.
Setting up the “Servo Dynamics Amplifier” can be done easily. The
amplifier has conveniently located potentiometer along the top edge

of the circuit board.

PRESET THE POTS. AS FOLLOWS - REFERENCE TO PAGE 7 FOR LOCATION
OF POTS AND TEST POINTS (TP)

Note: The adjustment pots. are 20 turns pots.
SIG. POT. - Mid range (10 Turns).

TACH. POT. - Turn fully CCW then 7 turns CW to approx. 7K OHMS
maximum as measured from TP1 to TP3 (use an ohm
meter set on the 20K scale) with the J1 connector removed.

CL. POT. - (Peak Current Limit) Turn Fully CCW
CMP. POT. - (Compensation) Turn fully CCW then 10 turns CW.
BALANCE POT.-Do not adjust at this time.

NOTE: Make sure a jumper is not installed between J1, Pin 6 and
Pin 7 if a regular analog tach is connected to Jl.

5) Apply power, but do not apply a signal to the amplifier.

6) Make sure the phase relationship of the commutator signals
are in the proper sequence with the motor wires to the motor.
(See “Appendix” section for detailed explanation). Otherwise
when you go to Step 7 you will observe very erratic motor
shaft rotation or the motor may appear to have dead spots at
various shaft positions.

7) To check tach or encoder phasing, slowly turn the current
limit pot. CW and observe the machine or motor shaft. If
the phasing of the motor and tach or encoder are correct, the
motor shaft will stay at rest or turn very slowly. If the
phasing is incorrect the machine/motor shaft will move
rapidly out of control and try to run away. Turn-off power
and reverse either the motor or (tach or Channel A and B of
the encoder) leads. Do not reverse both.

8) Turn the power back “ON” and observe that the system is now
under control.

19
Note: If the motor shaft continues to rotate slowly as the current
limit pot, is increased, there may be an offset problem which can be
adjusted by the Balance Pot. You may adjust the Balance pot now if
you desire to stop the motor from rotating.

9) Adjust the RMS pot. (R90) for the desired level of RMS
Limiting. Fully CCW is minimum RMS current limiting. Fully
CW is maximum RMS current limiting.

SEE APPENDIX SECTION ON HOW TO SET THE RMS POT.

10) To set the peak current limit pot (R92 A 20 TURN POT) for the
desired peak current. SEE TABLE BELOW.

11) Turn the CMP (compensation) pot R93 CW until the motor shaft
starts to oscillate at a high frequency rate, then turn the
CMP pot CCW until the oscillation stops -- then turn 1 turn
ccW for some additional safety margin.

Note 1: When using the encoder as velocity feedback and the shaft is
not turning there is a transport lag associated with the velocity
information. The only time information is present is when an
encoder’s line edge is transitioning. Hence the motor shaft may
still behave in an erratic manner after the tach pot has been turned
1 turn CCW after the motor shaft stops oscillating. Therefore it
maybe necessary to turn the compensation pot an additional 1/2 to 1
turn CCW.

See appendix section for more information about encoders.

Note 2: Sometimes a system may not break into a high frequency
oscillation when the CMP pot is fully clockwise, this may be normal
if the particular system the amplifier is connected to has a very
high mechanical resonant frequency and the maximum bandwidth range of
the amplifier is such that the resonance is not reached.

Note 3: Sometimes a system may continue to oscillate at a_high
frequency even if the CMP pot is fully CCW see notes in “System
Compensation using an Oscilloscope” for suggestions.

12) Now apply a small step signal to the amplifiers signal input
and observe how the motor shaft turns.

Does the motor shaft overshoot (under damped) when commanded to stop?
Does the motor shaft stop slowly (over damped) when commanded to stop?

Note: When observing the motor shaft or machine during a step input
signal. Observe the motor shaft just as the motor shaft stops?
20
13) Turn the tach pot. CCW for getting rid of overshoots.
Turn the tach pot. CW for making the system settle quicker.

Note: As a general rule it is advisable not to turn the tach. pot, to
a value greater than (7K), as measured with an OHM Meter from TP1 to
TP3 with the Jl connector removed.

14) After the proper velocity loop response is obtained, do not
adjust the tach pot any more.

15) Run the system at high speed and listen to it. Does it run
rough? If so turn “CMP” pot, a little CCW.

16) Adjust the signal pot. CW or CCW for maximum desired machine
velocity -- not the tach pot.

Example: A 10 volt input signal may be required to run the
machine at 200 inches per minute.

17) It may be necessary to readjust the balance pot for zero
drift or balance.

18) No further adjustments are necessary.

19) Now that you have successfully set up this axis. Let’s make
your adjustments a lot easier for the next amplifier that
you will be set up on the same axis of the same kind of machine.

20) Shut-off the “Power”.
21) Disconnect the J1 connector.

22) Using an OHM meter, set on the 20K OHMS Scale, connect one
end of the meter to TP1l which is common, and measure the
resistance at each test point. TP2 through TP7 and record
these values in the back of this manual, for future adjustments.

23) Install the J1 connector and apply power.

24) If you are going to use the Servo system in a position loop,
it may be necessary for the motor to turn CW or let’s say
the machine must go to the right with a positive signal command.

If the motor or machine is going in the wrong direction, the
easiest way to correct this is by switching the signal input
wires on the differential input command.

Note: One can not flip the signal command when the signal input
is set up for single-ended operation. It is necessary to
change the motor, commutator, and tach. wires to have
proper direction.

However, in a brushless system because of the 3 commutator wires and 3
motor wire combinations, the flipping of motor, commutator and tach. wires
turns out to be difficult to keep the combinations in correct sequence.
Using the differential command signal inputs makes reversing the motors’
shaft direction a lot easier.

MODIFIED COMPENSATION

Before modifying the amplifier compensation values, one should
try to anticipate what is going to happen when a particular part
value is changed.

IT IS ALSO NECESSARY TO TURN THE “CMP” POT FULLY CCW AND SLOWLY
TURN IT CW AFTER ANY COMPENSATION PART VALUE IS CHANGED.

It is also a good practice before changing any compensation
values to check for mechanical problems such as backlash,
misalignment, damaged couplings, lost motion, lead screw binding
or machine wind up.

1) UNABLE TO STOP THE SYSTEM FROM OSCILLATING AT A HIGH
FREQUENCY EVEN THOUGH THE “CMP” POT FULL CCW.

This type of system, is due to the ratio of the gains and break
frequencies, has too much bandwidth. The amplifier may be
exciting system resonances or the motor may be highly inductive,
typically 9 MHY or greater.

Note: Motors of 2 and 3 inch diameters, with torque constants
(Kt) of 2 in/lb amp or greater, are typically high inductive
motors. First increase value of C41 to .033 mfd. Try again if
system still oscillatory.

Increase the value of resistor R113 to 33K.

If the system oscillation is due to a mechanical resonance it may
be necessary to change the resistor R112. If the resonance is
above 700 HZ increase value of R112 to 20K.

2) IF THE MACHINE IS HAVING TROUBLE NULLING OR COMING INTO
POSITION, DUE TO NOT ENOUGH GAIN:

Increase C40 to .1 mfd and decrease R115 to 2K, and turn the tach
pot “CW” to increase the tach gain, of course, you will have to
turn the signal pot “CW”.

You can also increase the internal forward loop gain of the Servo
Amplifier by decreasing the value of R110 from 75 ohms to 30
ohms, however the amplifier is subject to greater drift.

3) A HIGH INERTIA SYSTEM WHICH USUALLY HAS A LOW MECHANICAL
BREAK FREQUENCY.

It may be necessary to increase C40 to .1 mfd and decrease R115
to 2K.

SHUNT REGULATOR

In some applications, where a motor decelerates with a lot of
inertia, or the motor is running at high speed and the motor is
commanded to stop. The systems kinetic energy being regenerated
is greater than the losses in the system hence, electrical energy
is pumped back into the main bus capacitor to charge up to a
voltage level beyond the capability of the amplifier, and will
cause the amplifier to turn off.

The amplifier is protected against this over-voltage condition
and will turn off and 210 VDC. To avoid nuisance tripping, the
amplifier may have a shunt regulator connected to the DC bus
capacitor. The shunt regulator turns on a 196 VDC and shunts
excessive voltage, through voltage dropping resistor(s), mounted
on the amplifier chassis. The shunt regulator is protected
against excessive dissipation by a series fuse, plus each
regulator transistor is fused individually.

ENCODER TO TACH SECTION

Some comments that were mentioned in previous sections of this
manual about Encoder to Tach topics may be repeated here for the
purpose of bringing the subject of the Encoder to Tach together
in one section.

GENERAL

The Encoder to Tach is also referred to as a Frequency to Voltage
Converter and can be used to create a tachometer signal from a
Square wave encoder.

For maximum speed of the Encoder to Tach circuit the encoder
should operate from a +5 Volt source and have a square wave line
driver output. However the Encoder to Tach circuit can work with
a single ended output encoder with pull up resistors located
somewhere in the encoders’ wiring.

Channels A & B and encoder common are connected to the J4 input
and a wire from Jl, Pin 6 to Jl, Pin 7 must be installed when the
encoder is connected to the J5 input.

The Encoder to Tach circuitry does a times 4 on the line count of
the encoder.

The Encoder to Tach module will provide a very low output ripple
voltage due to the unique feature which allows for equally spaced
output pulses (at any given speed) even if the quadrature of the
Encoder is not 90 degrees.

Low audible motor noise (because of the encoder’s pulse train)
results from a set of active filters in the Encoder to Tach
section.

The Encoder to Tach block shown in Figure #6 contains signal
conditioning networks and has 2 poles at approximately 332 CPS.

The maximum pulse rate of the Encoder to Tach is set by output
nonlinearity. The output scaling of 1.7 V/1000RPM has been
verified to be linear with a 2500 Line encoder with Line Driver
outputs up to 3000 RPM.

WIRING

The encoder wires Channel A, B, and Common must all be treated as
a digital data cable. It is most important that the 3 wires be
kept together and be in a shielded cable. The shield must float
at the amplifier end otherwise unnecessary noise will enter the
amplifier on J4 and erratic motor operation will occur.

Also noise pickup on the cable can go the other direction and get
into the controller.

J4 Pins 2 and 4 are to be operated from a 5 Volt encoder. These

2 input pins are connected to level translators located in the
amplifier.
