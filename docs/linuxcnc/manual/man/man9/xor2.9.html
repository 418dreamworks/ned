NAME

xor2 - Two-input XOR (exclusive OR) gate

SYNOPSIS

loadrt xor2 [count=N|names=name1[,name2…​]]

DESCRIPTION

The out pin is computed from the value of the in0 and in1 pins according to the following truth table:
