LED_DIM(9) Manual Page

NAME

led_dim - HAL component for dimming LEDs

SYNOPSIS

loadrt led_dim [count=N|names=name1[,name2…​]]

DESCRIPTION

Component for LED dimming according to human perception of brightness of light. .LP The output is calculated using the CIE 1931 formula.

FUNCTIONS

led-dim.N

Update the output value

PINS

led-dim.N.in float in

Brightness input value → 0 to 1

led-dim.N.out float out

Luminance output value → 0 to 1
