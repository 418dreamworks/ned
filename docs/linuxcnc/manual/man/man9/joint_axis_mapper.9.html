JOINT_AXIS_MAPPER(9) Manual Page

NAME

joint_axis_mapper - Translate faults from Joint to Axis

SYNOPSIS

loadrt joint_axis_mapper

DESCRIPTION

By default joint.n.amp-fault-in triggers an error message Joint n amplifier fault.

This component is a translation layer that shows an additional message {L} Axis fault detected, with {L} being the axis letter associated with the faulted joint.

Usage example: loadrt joint_axis_mapper coord=xyz addf joint_axis_mapper servo-thread

net x-fault joint.0.faulted ⇒ jam.x-fault net y-fault joint.1.faulted ⇒ jam.y-fault net z-fault joint.2.faulted ⇒ jam.z-fault

FUNCTIONS

PINS
