tool_mmap_read(1) Manual Page

NAME

tool_mmap_read - A component of the tool database system (an alternative to the classic tooltable)

DESCRIPTION

tool_mmap_read is not intended to be invoked by the user.

SEE ALSO

linuxcnc(1)*

See the Tool Database Interface section of the LinuxCNC Documentation for more information at https://linuxcnc.org/docs/devel/html/tooldatabase/tooldatabase.html.

Much more information about LinuxCNC and HAL is available in the LinuxCNC and HAL User Manuals, found at /usr/share/doc/LinuxCNC/.
